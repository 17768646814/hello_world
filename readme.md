## 初始化模块管理工具
```
mkdir hello_world
cd hello_world
npm init
```
> npm init后按照提示输入信息，例如
```
hello_world
1.0.0
index.js
webpack
https://github.com/17768646814/hello_world.git
webpack react
pch
ISC
```
最后选择输入yes，回车，最终生成package.json

npm i --D colors

npm i -S react react-dom react-router@3.x
#markdown js & antd
npm i -S markdown antd
